## üöÄ How to Run Locally

This is a single-file app ‚Äî just run the `index.html`!

### Prerequisites
- A modern browser (Chrome, Firefox, Edge)
- **Visual Studio Code** (recommended)
- **Live Server** VS Code extension

### Setup Instructions
1. Clone this repository or download `index.html`.
2. Open it in **VS Code**.
3. Right-click and choose **"Open with Live Server"** or click the **"Go Live"** button.
4. In your browser:
   - Enter a company name (e.g., `"Tesla"`) ‚Üí Click **Analyze**
   - Click **"‚ú® Generate AI Analysis"** for insights
